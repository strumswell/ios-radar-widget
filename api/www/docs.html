API Documentation (preliminary)

Short description of all API routes and all its options.

Get Mapshot

/api/v2/mapshot?lat=52.41&lon=12.55&map=3&color=8&key=KEY

Cached: TRUE, 10 minutes

Options

Obligatory:
  • lat: latitude of position
  • lon: longitude of position
  • key: your api key
Optional:
  • map: map type
    • 1: Original
    • 2: Universal Blue
    • 3: TITAN
    • 4: The Weather Channel
    • 5: Meteored
    • 6: NEXRAD Level III
    • 7: Rainbow @ SELEX-IS
    • 8: Dark Sky (standard)
  • color: color of rain radar
    • 1: hybrid map (OpenStreetMap)
    • 2: hybrid map (CartoDB)
    • 3: dark map (ArcGIS)
    • 4: satellite map (ArcGIS, standard)
    • 5: hybrid map (ArcGIS)
